Projet 1 : Site vitrine en HTML/CSS/JS

  • ⬅ Retour au portfolio

Présentation

Ce projet est un site vitrine simple réalisé uniquement avec HTML, CSS et JavaScript. Il a pour but de montrer la structure d’une page web et l’utilisation des bases du développement frontend.

Fonctionnalités

  • Structure HTML sémantique
  • Stylisation avec CSS (mise en page responsive, couleurs, typographie)
  • Effets visuels avec JavaScript (animations et interactions)

Code utilisé

Voici un petit exemple de code issu de ce projet :

<header>
  <h1>Bienvenue sur mon site vitrine</h1>
  <nav>
    <a href="#services">Services</a>
    <a href="#contact">Contact</a>
  </nav>
</header>

Conclusion

Ce projet m’a permis de mieux comprendre la structure d’un site web et de pratiquer les bases du HTML, CSS et JavaScript.
