Présentation

Ce projet est une petite application web qui récupère des données à partir d’une API externe et les affiche à l’utilisateur. L’objectif était de pratiquer les appels réseau et la manipulation de données JSON.

Fonctionnalités

  • Appel d’une API avec fetch()
  • Affichage dynamique des données reçues
  • Interface responsive et facile à utiliser

Exemple de code

fetch("https://api.exemple.com/data")
  .then(response => response.json())
  .then(data => {
    console.log(data);
    // Ici, on affiche les infos dans la page
  })
  .catch(error => console.error("Erreur :", error));

Conclusion

Grâce à ce projet, j’ai appris à interagir avec une API, traiter les données reçues et les intégrer dans une page web.
