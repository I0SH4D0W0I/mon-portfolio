Projet 3 : Mini-jeu en JavaScript

Présentation

Ce projet est un mini-jeu codé en JavaScript. Il permet de mettre en pratique les notions de logique, d’événements et de DOM.

Fonctionnalités

  • Gestion des événements clavier/souris
  • Interaction avec le DOM (score, affichage des éléments)
  • Ajout d’animations simples

Exemple de code

// Exemple simple : jeu du clic rapide
let score = 0;
const btn = document.getElementById("playBtn");

btn.addEventListener("click", () => {
  score++;
  document.getElementById("score").textContent = "Score : " + score;
});

Conclusion

Ce projet m’a permis de progresser en JavaScript en créant une petite expérience ludique et interactive.
